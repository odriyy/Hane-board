The website is built using an index.html template. 
Each NFC tag is mapped to a specific HTML page, where the tag’s unique ID corresponds to a designated page through a matching code system.

The center square (menu canvas) has a resolution of 340 × 240 pixels. 
The larger squares are currently designed at 257 × 257 pixels, its recommended to create them at a higher resolution first and then scale them down to maintain better image quality
